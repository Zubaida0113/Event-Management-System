    What is this website for?

    This website is a platform for event organizers to create, manage, and promote their events, and for attendees to discover and register for events.

    How do I create an account?

    You can create an account by clicking on the "Sign Up" or "Register" button and filling in the required information.

    How do I know if an event is still happening?

    Event Finder displays the most up-to-date information about events, including any cancellations or changes.

    How do I register for an event?

    You can register for an event by clicking the 'Register' button on the event page and filling out the required information.
